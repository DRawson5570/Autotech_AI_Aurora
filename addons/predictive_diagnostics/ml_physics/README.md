# ML Physics Diagnostic Engine

This module implements the ML-based approach to physics-informed automotive diagnostics.

## Approach

Instead of hardcoding physics equations, we train a neural network to **learn** the physics
relationships from synthetic data generated by physics simulators.

## Comparison to Deterministic Physics

| Deterministic (`../physics/`) | ML Physics (`./`) |
|------------------------------|-------------------|
| Explicit equations | Learned relationships |
| One system at a time | Unified multi-system model |
| Perfect explainability | Attention-based explanation |
| Manual fault encoding | Learns from synthetic data |
